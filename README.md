### 使用releases                                                                   
1. 安装[Node.js](https://nodejs.org/)
2. 下载[bilive_client.zip](https://github.com/lzghzr/bilive_client/releases/latest)
3. 解压并定位到目录
4. 命令行 npm install
5. 命令行 npm start

### 自行编译
1. 安装[git](https://git-scm.com/downloads)
2. 安装[Node.js](https://nodejs.org/)
3. 命令行 git clone
4. 命令行 cd bilive_client
5. 命令行 npm install
6. 命令行 npm run build
7. 命令行 npm start
